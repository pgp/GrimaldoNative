# GrimaldoNative
A client/server console application using challenge-response authentication for unlocking screen remotely on Linux PCs


#### Main dependencies

- [argparse](https://github.com/jamolnng/argparse) for command line argument parsing
- [SphincsPlus](https://github.com/sphincs/sphincsplus) as signature scheme for building challenge-response authentication
- [mongoose](https://github.com/cesanta/mongoose) for implementing client-server logic
